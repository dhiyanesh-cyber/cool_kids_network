# Cool Kids Network - Frontend

## Project Setup

### Prerequisites
- Node.js (v16+)
- npm or yarn

### Installation
1. Clone the repository
2. Navigate to the frontend directory
3. Install dependencies:
   ```
   npm install
   ```

### Running the Application
```
npm start
```

### Building for Production
```
npm run build
```

### Key Features
- User Registration
- Character Generation
- Role-based Access Control
- User Profile View
- User Listing

### Technical Stack
- React
- React Router
- Tailwind CSS
- Axios for API calls
